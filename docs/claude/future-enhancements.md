# CLM Future Enhancements

A running collection of ideas for future enhancements and features that don't yet have a home in a design doc, handover, or GitHub issue.

## Related documents

- **`docs/claude/TODO.md`** — known bugs, technical debt, and in-flight feature work. Items here are more concrete than this file.
- **`docs/claude/design/`** — active design documents. Features with a design doc live there, not here.
- **`docs/claude/*-handover.md`** — feature-specific handover docs for in-progress work (excluding `*-archive.md`).

## How to use this document

- Add ideas as they come up during development, code review, or user conversations.
- Each item should include a short motivation and a source (where the idea came from, so context is recoverable).
- Items **graduate out** of this doc when they move into a design doc, a handover, or a GitHub issue — delete them from here when that happens.
- Delete stale items rather than letting them accumulate indefinitely.
