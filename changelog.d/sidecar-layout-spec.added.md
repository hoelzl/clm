- Course specs may now set a per-course `<sidecar-layout>` (`subdir` / `sibling`) to control where newly written authoring sidecars land. With `subdir`, a build records a topic's first HTTP-replay cassette into the topic's `cassettes/` folder instead of next to the slides, even when that folder does not exist yet. Precedence: `CLM_SIDECAR_LAYOUT` env var, then `<sidecar-layout>`, then `[tool.clm] sidecar-layout`; reads still auto-detect both layouts.
